# HEXA-CMCA — Critical Mineral Conflict Arbitration (kick-spec)

> **One-line summary**: a peaceful critical-mineral dispute-resolution architecture where (i) royalty splits between sovereign / co-claimant / external-investor are derived from **Nash 1953 bargaining + Shapley 1953 trilateral value + Hotelling 1931 exhaustible-resource pricing + Coase 1960 externality bargaining** rather than political bid; (ii) recovery yields are bound by **D2EHPA solvent-extraction physics (Sastri-Shibata 2003) + 5-stage Kremser 1930 countercurrent**; (iii) compliance with arbitration awards is monitored via civilian **Sentinel-1 SAR InSAR** mining-activity change detection. Reference case: Ukraine-US 2025-04-30 Mineral Resources Agreement (55-mineral / 50-50 royalty / Joint Reconstruction Investment Fund). Reusable to Greenland / DRC / Mongolia / Chile / Indonesia / Australia indigenous-title cases.

> Honest scope per raw 91 C3: kick-spec at this revision is **theoretical-architectural** (Nash-bargaining solver + Hotelling price-path simulator + D2EHPA mass-balance). ## §0 WHY THIS IS A DOMAIN, NOT AN ESSAY

The 2026-05-06 contested-mineral research thread surfaced that **the resource layer is the only layer of post-fossil geopolitics with measurable, falsifiable, reproducible physical anchors**:

- narrative / identity / sovereignty-framing layers → political-science discourse, no physical-limit ceiling, no falsifier
- **resource layer** → USGS reserves data + LME spot + Hotelling price path + Nash bargaining + D2EHPA chemistry are **all empirically anchored and reproducible**

A peaceful arbitration architecture anchored at this layer **generalizes** across cases that today are resolved by political bid or unilateral seizure: Greenland REE (2025 US offer), DRC cobalt (multi-claimant SOE concessions), Mongolia copper (Oyu Tolgoi state-vs-Rio Tinto split), Chile lithium salar (state-vs-private royalty), Indonesia nickel (downstream-policy disputes), Australia indigenous-title royalty negotiations. One domain, N reusable peaceful-resolution cases.

→ qualifies for n6-arch: physical-anchored + falsifiable + reusable + cross-axis fan-in. Goal-state = a transparent royalty + recovery-yield + compliance-monitoring stack that lowers the political cost of negotiated settlement vs unilateral action.

## §9 FALSIFICATION OF THE KICK-SPEC ITSELF

This kick-spec is **wrong** if any of the following hold:

1. ≥ 1 of the 6 precursor domains scores alien_grade < 7 (would require uplift before fan-in)
2. The Nash bargaining model is structurally inapplicable to trilateral (sovereign + co-claimant + investor) settings (Shapley value fallback documented but unverified)
3. D2EHPA chemistry from amd-ree-mineshaft-phes does not generalize to spodumene / laterite / hard-rock feed matrices — different mineralogy may invalidate β_Y/Eu transfer
4. Sentinel-1 12-day revisit + 5 m IW resolution is insufficient to detect extraction-rate changes at the relevant scale (would require commercial SAR or higher-revisit constellation)
5. No enforced ICSID / UNCITRAL precedent exists in the past decade for mineral-rights arbitration awards against a non-cooperative sovereign — peaceful enforcement pathway would need to be re-architected

Each of (1)-(5) maps directly to F-CMCA-MVP-1..5 above.
