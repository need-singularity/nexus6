# v4 loop 8 — M3 case 4b(i): Theorem B n = 2·q^b Lean4 formal

## Target

For all n = 2·q^b with q odd prime ≥ 3, b ≥ 2:
  σ(n)·φ(n) ≠ n·τ(n)

## Demonstration structure (Lean4 `theorem_B_2_prime_pow`)

### 1. Multiplicative decomposition (gcd(2, q^b) = 1)

- σ₁(2·q^b) = 3·σ₁(q^b)
- φ(2·q^b) = φ(q^b)
- τ(2·q^b) = 2·(b+1)

### 2. Reconstruction of σ(q^b)·φ(q^b)

Reuse case 3 geometric sum identity:
  (q-1)·Σ_{k=0..b} q^k = q^(b+1) - 1
  σ(q^b)·φ(q^b) = q^(b-1)·(q^(b+1) - 1)

### 3. Core inequality reduction

- σφ(n) = 3·q^(b-1)·(q^(b+1) - 1)
- nτ(n) = 4·q^b·(b+1) = q^(b-1)·(4q(b+1))

Cancel q^(b-1) > 0:
  equality ⟺ 3·(q^(b+1) - 1) = 4q(b+1)
          ⟺ 3·q^(b+1) = 4q(b+1) + 3

### 4. Key inequality (`key_ineq_4bi`)

∀ q ≥ 3, b ≥ 2 : 3·q^(b+1) > 4q(b+1) + 3

**Demonstration**: Nat.le_induction on b from 2.

Base b=2: 3·q³ > 12q + 3.
  q ≥ 3 → q² ≥ 9 → q³ = q·q² ≥ 9q
  9q > 4q + 3 (trivial for q ≥ 1)

Step b → b+1 (b ≥ 2):
  ih: 3·q^(b+1) > 4q(b+1) + 3
  3·q^(b+2) = q·(3·q^(b+1)) > q·(4q(b+1) + 3) = 4q²(b+1) + 3q
  Need: 4q²(b+1) + 3q ≥ 4q(b+2) + 3
  diff = 4q(b+1)(q-1) - q - 3 ≥ 12q² - 13q - 3 > 0 for q ≥ 3

### 5. Conclusion

From equality assumption we derive 3·q^(b+1) = 4q(b+1) + 3 → contradiction with key inequality → σφ ≠ nτ.

## Relation to Case 2a

- Case 2a (n = 2q): b = 1. Equality at q = 3 (n=6), inequality for q ≥ 5 (TheoremB_Case2_P2).
- Case 4b(i) (n = 2·q^b): b ≥ 2. Inequality for every q odd prime ≥ 3.

So q=3, b=1 is the unique equality solution → this is n=6.
