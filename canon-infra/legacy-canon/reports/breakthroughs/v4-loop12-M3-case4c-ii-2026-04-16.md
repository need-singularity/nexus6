# v4 loop 12 — M3 case 4c(ii): Theorem B n = 2^a·q·r Lean4 formal

## Target

For all n = 2^a·q·r with a ≥ 2 and q ≥ 3, r ≥ 5 odd distinct primes:
  σ(n)·φ(n) ≠ n·τ(n)

## Demonstration strategy — product of STRICT bounds

### Core idea

Multiply two individual strict bounds:
- σφ(2^a) > 2^a·(a+1)  STRICT for a ≥ 2  (`sigma_phi_2pow_strict`)
- σφ(qr) > qr·4         STRICT for q,r odd distinct ≥ 3  (`sigma_phi_qr_strict`)

Multiplicative decomposition: σφ(2^a·q·r) = σφ(2^a)·σφ(qr)
nτ likewise: nτ(2^a·q·r) = 2^a·(a+1) · qr·4

Positive integer product: A > A', B > B', A', B' > 0 → AB > A'B'.
Therefore σφ(n) > nτ(n) STRICT → σφ ≠ nτ.

### `sigma_phi_2pow_strict` demonstration

σφ(2^a) = (2^(a+1) - 1) · 2^(a-1) (geom_sum + totient_prime_pow)
RHS = 2^a·(a+1) = 2·2^(a-1)·(a+1)

Use identity 2^(a+1) = 4·2^(a-1) (let X := 2^(a-1)):
  (4X - 1)·X > 2X·(a+1)

`prime_pow_strict_gt` (case 3): 2^(a+1) > 2(a+1)+1, i.e., 4X > 2a+3.

Multiply: 4X² ≥ (2a+4)X, diff = 4X² - X ≥ 2aX + 3X > 2aX + 2X = 2X(a+1). ✓

### `sigma_phi_qr_strict` demonstration

σ(qr)·φ(qr) = (q+1)(r+1)(q-1)(r-1).

Auxiliary bounds:
- (q+1)(q-1) = q² - 1 ≥ 2q + 2  (for q ≥ 3, from (q-1)² ≥ 4)
- (r+1)(r-1) ≥ 2r + 8            (for r ≥ 5, from (r-1)² ≥ 16)

Product: (q+1)(q-1)·(r+1)(r-1) ≥ (2q+2)(2r+8) = 4qr + 16q + 4r + 16 > 4qr. ✓

## Generalization potential

This demonstration pattern (product of individual strict bounds) extends to **case 4c general**:
- ω(n) ≥ 3: n = p₁^{a₁}·...·p_k^{a_k}
- For each prime power, the strict bound σφ(p^a) > p^a·(a+1):
  - p = 2, a = 1: σφ(2) = 3 < 4 = 2·2. **Opposite direction!** (σφ < nτ)
  - p = 2, a ≥ 2: STRICT >
  - p ≥ 3, a ≥ 1: STRICT >
- When 2 ∥ n (v_2 = 1): σφ(2) = 3 < 4. But the remaining ω ≥ 3 product overrides with >.
- This mixed case is the trickiest — requires Lean4 formalization.
