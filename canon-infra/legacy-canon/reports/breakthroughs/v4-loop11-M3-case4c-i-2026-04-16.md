# v4 loop 11 — M3 case 4c(i): Theorem B n = pqrs Lean4 formal

## Target

For all n = pqrs with p ≥ 2, q ≥ 3, r ≥ 5, s ≥ 7 distinct primes:
  σ(n)·φ(n) ≠ n·τ(n)

Formalization of the **ω(n) = 4, squarefree** region — 4-prime extension of case 4a (pqr).

## Demonstration structure (Lean4 `theorem_B_four_primes`)

### 1. Multi-coprime construction

- `coprime_triple_prod_prime`: (pqr).Coprime s via two applications of `coprime_mul_iff_right`
- sigma/totient/tau are all multiplicative; reuse `sigma_one_pqr` / `totient_pqr` / `tau_pqr` from case 4a

### 2. Formulae

- σ(pqrs) = (p+1)(q+1)(r+1)(s+1)
- φ(pqrs) = (p-1)(q-1)(r-1)(s-1)
- τ(pqrs) = 16

### 3. Two key inequalities

**Lower bound** (etc):
  (p-1)(q-1)(r-1)(s-1) ≥ 1·2·4·6 = 48  
  Demonstration: stepwise `Nat.mul_le_mul` chain (p-1 ≥ 1, q-1 ≥ 2, r-1 ≥ 4, s-1 ≥ 6)

**Strict gt** (σ-bound):
  (p+1)(q+1)(r+1)(s+1) > pqrs  
  Demonstration: (p+1)(q+1) > pq AND (r+1)(s+1) > rs, product via `Nat.mul_lt_mul`

### 4. Combined — σφ > 16pqrs = nτ

  σφ(n) = (p+1)(q+1)(r+1)(s+1) · (p-1)(q-1)(r-1)(s-1)
        ≥ (p+1)(q+1)(r+1)(s+1) · 48            [lower bound]
        > pqrs · 48                             [strict gt]
        > pqrs · 16                             [48 > 16, pqrs > 0]
        = nτ(n)

Contradiction from equality assumption → QED.
