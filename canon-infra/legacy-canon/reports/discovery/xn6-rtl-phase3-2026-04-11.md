## 3. Six primitive-op module summary

### 3.1 Omega_1 GEMM — `gemm_core12_00`

- 12x12 systolic array = sigma^2 = 144 MAC cells
- `genvar gi, gj` double loop unrolls M*N cells
- sigma=12 stage `valid_pipe` pipeline
- Built-in `n6_assert` output — `sigma*phi=n*tau=24` runtime hardware assert

### 3.2 Omega_2 SOFTMAX — `softmax_phi2_exp`

- 4-stage chain built in: exp (Phi6 `x^2-x+1`) -> max reduce -> sub -> norm
- VARIANT 2'b00~11 selects stage
- phi=2 * 4 stages = tau*phi=8 cycles (7x savings vs conventional GELU 14 cycles)
- Egyptian {1/2, 1/3, 1/6} reciprocal LUT annotations inserted

### 3.3 Omega_3 TOPK — `topk_phi2_pick`

- k=phi=2 top-2 selection (primary for MoE active experts)
- sigma=12 candidates, IDX_BITS=4 (log_2 sigma)
- 2-phase FSM: find top-1 -> exclude top-1 and find top-2
- cycles = phi=2

### 3.4 Omega_4 GATE — `gate_mu1_and`

- mu=1 single-cycle combinational gating
- VARIANT: mul (00) / and (01) / or (10) / xor (11)
- Boltzmann 1/e threshold (`BOLTZ_INV_E = 8'd94`)-based zero-skip `keep_bits` emitted in parallel
- 63% MAC saving rationale (1/e threshold distribution)

### 3.5 Omega_5 REDUCE — `reduce_tau4_sum`

- tau=4 inputs -> 1 output (sum/max/min/mean)
- 2-stage pairwise reduce tree: depth = log_2(tau) = phi=2
- mean stage is `>> 2` (tau=2^phi shift)
- output width `WIDTH_BITS*2` (overflow handling for sum)

### 3.6 Omega_6 CONV6 — `conv6_3x3`

- 3x3 spatial conv, tau=4 channel parallelism, sigma=12 image
- 4-deep loop c/y/x/ky*kx (synth unrolls to MAC blocks)
- VARIANT switches 3x3 / 1x1 / DW / PW
- n=6 cycle pass (3x3), sigma*tau=48 MAC (dw)

## 6. Core-identity verification

```
sigma*phi = 12*2 = 24
n*tau     = 6*4  = 24
J_2       =       24         <- unique solution
MAC       = sigma^2 = 144    (Omega_1 GEMM)
SOFTMAX   = tau*phi = 8 cycles (Omega_2)
TOPK      = phi = 2 cycles   (Omega_3)
GATE      = mu = 1 cycle     (Omega_4)
REDUCE    = log_2(tau) = phi=2 (Omega_5)
CONV6     = n = 6 cycles     (Omega_6)
Sum cycles = 1+2+2+6+8+12 = 31  (sigma^2 under-score)
```

Each primitive's cycle count is composed only of sub-factors of `sigma*phi=n*tau=24` (1, 2, 4, 6, 8, 12).
